# Aula 08 — Estruturas de dados (map / filter / forEach) e objetos

Nesta aula praticamos **arrays** e operações comuns do dia a dia (iterar, transformar e filtrar), além de reforçar **objetos** e **array de objetos**.

## Materiais

- Arquivos da aula:
  - [index.html](./index.html)
  - [main.js](./main.js)
  - [banco-de-dados.json](./banco-de-dados.json)

## Como executar

- Abra o arquivo [index.html](./index.html) no navegador (Chrome/Edge)
- Abra o DevTools:
  - Windows/Linux: `F12` ou `Ctrl + Shift + I`
  - Aba: Console

## Organização do script

O [main.js](./main.js) abre um menu com `prompt()` para executar **um tópico por vez**.

## Tópicos no menu

### 1) `push` + `for` (lista de frutas)

- Adiciona itens em um array usando `push()`
- Percorre repetindo com `for`

### 2) `map` (identificar frutas)

- Transforma um array em outro array
- Retorna strings identificadas por índice (par/ímpar)

### 3) `filter` + `forEach` (carros)

- Lista os carros com `forEach`
- Filtra apenas alguns valores com `filter`

### 4) Objetos (post e método)

- Cria um objeto `post` com propriedades e array de comentários
- Usa um método dentro do objeto (`curtir`)

### 5) Array de objetos (pessoas)

- Conta quantos são maiores de idade com `forEach`
- Usa `map` para “desativar” menores de idade
- Usa `filter` para remover itens “INATIVO”
