# Aula 11 — HTML/CSS (estrutura) + JavaScript no navegador

Nesta aula organizamos uma página HTML com `header`, `nav` e `main`, aplicamos CSS e adicionamos um JS simples para interagir com o conteúdo.

## Materiais

- Arquivos da aula:
  - [index.html](./index.html)
  - [main.css](./main.css)
  - [index.js](./index.js)

## Como executar

- Abra o arquivo [index.html](./index.html) no navegador (Chrome/Edge)
- Abra o DevTools:
  - Windows/Linux: `F12` ou `Ctrl + Shift + I`
  - Aba: Console

## O que foi praticado

- Estrutura de página com HTML semântico (`header`, `nav`, `main`)
- CSS básico via [main.css](./main.css)
- JS no navegador via [index.js](./index.js):
  - `prompt()` para receber um título
  - `document.querySelector()` para selecionar o `h1`
  - `textContent` para atualizar o texto na tela
