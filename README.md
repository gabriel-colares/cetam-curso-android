## Plano de ensino

- Documento completo: [plano-de-ensino-curso.md](./plano-de-ensino-curso.md)
- Curso: Qualificação Profissional **Desenvolvedor de Aplicações Móveis com Android**
- Eixo tecnológico: **Informação e Comunicação**
- Formato: **Presencial**
- Carga horária: **130h**
- Docente: **Gabriel Soares Colares**

**Objetivo (síntese):** capacitar o discente a desenvolver aplicações móveis para Android com tecnologias modernas e multiplataforma (JavaScript/React Native), cobrindo lógica, POO, UI, consumo de APIs e publicação.
